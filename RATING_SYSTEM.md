# 📊 评价体系说明（v1.2.0）

## 🎯 评价标准优化

### 背景
由于引入了**100 抽保底机制**，评价标准需要更加合理：
- 有保底的情况下，获得神话变得相对容易
- 评价应该区分"真正欧皇"和"保底战士"
- 综合考虑**欧气值**和**保底效率**

---

## 📈 评价维度

### 1. 欧气值（Luck Ratio）
```
欧气值 = 总价值 / (总抽数 × 600)
```
- 反映整体收益水平
- 600 是神话物品的价格（理论最大值）

### 2. 保底效率（Pity Efficiency）
```
保底效率 = 实际获得神话数 / 理论保底神话数
理论保底神话数 = floor(总抽数 / 100)
```
- 反映你是否超越了保底水平
- > 1 说明你靠运气抽到了额外神话
- < 1 说明你主要靠保底

## 💡 评价逻辑示例

### 例 1：真·欧皇
- 总抽数：200 抽
- 获得神话：5 个（2 个新 + 3 个重复）
- 理论保底：2 个（200/100）
- 保底效率：5/2 = 2.5
- 欧气值：假设 40%
- **评价**：太乙欧圣 ☯️

### 例 2：保底战士
- 总抽数：200 抽
- 获得神话：2 个（刚好保底）
- 理论保底：2 个
- 保底效率：2/2 = 1.0
- 欧气值：假设 18%
- **评价**：普通人 😐

### 例 3：非酋
- 总抽数：200 抽
- 获得神话：1 个（还没到保底）
- 理论保底：2 个
- 保底效率：1/2 = 0.5
- 欧气值：假设 8%
- **评价**：非酋 😭

1. **输入兑换码**：`主任真帅` 获得 100 抽
2. **观察保底**：前 60 抽看运气
