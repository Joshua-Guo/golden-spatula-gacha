# 📊 完整评分体系说明（v1.3.0 宽松版）

## 🎯 评价标准

### 双维度评价系统

评分基于两个维度：

1. **欧气值（Luck Ratio）**
   ```
   欧气值 = 总价值 / (总抽数 × 600)
   ```
   - 总价值：所有物品的价值总和
   - 600：神话物品价格（理论最大值）

2. **保底效率（Pity Efficiency）**
   ```
   保底效率 = 实际获得神话数 / 理论保底神话数
   理论保底神话数 = floor(总抽数 / 100)
   ```
   - 反映你是否超越了保底水平
   - > 1 说明靠运气抽到了额外神话
   - < 1 说明主要靠保底

## 📈 评价示例

### 例 1：普通玩家
- 总抽数：100 抽
- 获得神话：1 个（保底）
- 总价值：约 800 石头
- 欧气值：800 / (100 × 600) = 1.33%
- **评价**：超级大非酋 💀

### 例 2：运气不错
- 总抽数：100 抽
- 获得神话：2 个（1 个保底 + 1 个运气）
- 总价值：约 1400 石头
- 欧气值：1400 / 60000 = 2.33%
- 保底效率：2 / 1 = 2.0
- **评价**：超级非酋 💔（欧气值不够）

### 例 3：真欧皇
- 总抽数：100 抽
- 获得神话：3 个（1 个保底 + 2 个运气）
- 总价值：约 2000 石头
- 欧气值：2000 / 60000 = 3.33%
- 保底效率：3 / 1 = 3.0
- **评价**：非酋 😭（欧气值还是不够）

### 例 4：超级欧皇
- 总抽数：200 抽
- 获得神话：6 个（2 个保底 + 4 个运气）
- 总价值：约 4000 石头
- 欧气值：4000 / 120000 = 3.33%
- 保底效率：6 / 2 = 3.0
- **评价**：非酋 😭

> ⚠️ **注意**：即使保底效率很高，如果欧气值不够，评价也不会高。这是因为爆率提高后，理论最大值也提高了。

### 例 5：完美欧皇
- 总抽数：100 抽
- 获得神话：5 个（全是运气，远超保底）
- 总价值：约 3500 石头
- 欧气值：3500 / 60000 = 5.83%
- 保底效率：5 / 1 = 5.0
- **评价**：小非 😅

## 💡 评价机制说明

### 为什么要求看起来还是很高？

虽然评价标准已经大幅降低，但由于**爆率提高**，理论上应该更容易获得高价值物品。

**v1.3.0 爆率：**
- 神话：0.50%（平均每 200 抽 1 个）
- 史诗：1.00%（平均每 100 抽 1 个）
- 特殊：2.00%（平均每 50 抽 1 个）
- 稀有：8.00%（平均每 12.5 抽 1 个）

**保底机制：**
- 每 100 抽必出 1 个神话
- 实际神话获取率 ≈ 1%（保底 + 概率）

### 如何获得高评价？

1. **输入兑换码**：`主任真帅` 获得 100 抽
2. **利用保底**：100 抽内必出神话
3. **多次兑换**：继续输入兑换码获得更多抽数
4. **累积统计**：抽数越多，统计越准确
5. **超越保底**：真正的欧皇会在保底前就出神话
